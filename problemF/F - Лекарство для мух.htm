Входной файл: input.txt

Выходной файл: стандартный поток вывода («вывод на экран»)

Ограничение по времени: 1 секунда на тест

Ограничение по памяти: 32 Мб

Условие задачи

В лаборатории, занимающейся разработкой лекарств, произошла чрезвычайная ситуация. Все подопытные мухи заболели: кто-то ангиной, кто-то ветрянкой, а кого-то радикулит замучил. В лаборатории производится множество лекарственных препаратов от разных болезней, и имеется по одному опытному образцу каждого препарата, который может излечить некоторые болезни, которыми заболели мухи.

Помогите лаборанту Пете определить, какое максимальное количество мух он сможет вылечить с помощью имеющегося набора лекарств. Каждого лекарства хватит на излечение только одной мухи.

Входные данные

Первая строка файла input.txt содержит два числа N (1 <= N <= 100) и K (1 <= K <= 100), где N - количество мух, а K - количество лекарств. Далее следует K строк, описывающих имеющиеся лекарства. Каждая строка начинается числом M (1 <= M <= N), определяющим количество мух, которых можно излечить данным лекарством. Далее в строке идёт M чисел, указывающих номера этих мух.

Выходные данные

В стандартный поток необходимо вывести число, показывающее максимальное количество мух, которое можно вылечить.

input.txt

Стандартный поток вывода

3 3

1 1

2 1 3

1 3

2
