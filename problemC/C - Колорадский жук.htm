Входной файл: input.txt

Выходной файл: стандартный поток вывода («вывод на экран»)

Ограничение по времени: 3 секунды на тест

Ограничение по памяти: 32 Мб

Условие задачи

Летние каникулы Петя проводит у бабушки в деревне и помогает ей ухаживать за картофелем, который они посадили весной. Этим летом картофель уродился на славу, и на грядку прилетела целая стая колорадских жуков. Для того чтобы уничтожить жуков, Петя с помощью своего квадрокоптера провёл разведку и определил на каких картофельных кустах сидят жуки. После этого Петя прикрепил к квадрокоптеру банку со спреем "Антижук" и решил распылить её в месте скопления наибольшего количества жуков. Помогите Пете найти это место и посчитать какое максимальное количество жуков попадут в зону действия спрея.

Входные данные

Первая строка файла input.txt содержит два числа: целое число N (1 <= N <= 100) и вещественное число R (0.01 <= R <= 1000.00) с двумя знаками дробной части, где N - количество жуков на грядке, R - радиус действия спрея от места его распыления (все жуки, находящиеся на расстоянии не превышающем R от места распыления спрея, попадают под его действие). Далее следует N строк, задающих координаты жуков X и Y (0 <= X, Y <= 100). X и Y - целые числа.

Выходные данные

В стандартный поток необходимо вывести целое число, равное максимальному количеству жуков, которые могут попасть под действие спрея.

input.txt

Стандартный поток вывода

3 2.00

0 0

0 3

5 5

2
